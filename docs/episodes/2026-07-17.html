Guardrails can't tell a defender from an attacker

Self-Attention — 2026-07-17

2026-07-17

Hugging Face disclosed an intrusion run end to end by an autonomous agent framework — and the detail that matters is what happened next: the frontier models they reached for to do forensics refused, because a request to analyze an intrusion is indistinguishable from an intrusion. IBM gave away a trillion tokens of synthetic code paired with execution traces, on the thesis that GitHub teaches a model what code looks like and never what it does. And Moonshot's Kimi K3 lands around third on an outside index with its weights still ten days away.
