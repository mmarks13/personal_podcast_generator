What OpenAI actually spends, and the plumbing the agents run on

Self-Attention — 2026-06-18

Daily AI Podcast — Thursday, June 18, 2026

A quiet day for new models, a loud one for everything around them. Leaked financials show OpenAI losing roughly $21B against ~$13B revenue just as it heads toward a public filing; at the G7, Anthropic's Dario Amodei and DeepMind's Demis Hassabis pitched a U.S.-led AI coalition while the U.S. held off blacklisting DeepSeek. Underneath the headlines, the agent stack kept building out — runtime skill discovery and its attack surface, deployment-simulation evals, and an agent-native code host doing 22 commits a second.

Papers

  • Agentic Resource Discovery (ARD) — Hugging Face — open draft spec (contributors from Microsoft, Google, GoDaddy, HF) for runtime discovery of tools, skills, and MCP servers via a static ai-catalog.json manifest plus a dynamic search API.
  • PhantomSkill: Malicious Code Injection in Agent Skill Ecosystems — hides malicious behavior in a skill's auxiliary resources (not its description) via VulMask, which rewrites overt malicious scripts into vulnerability-shaped code triggered under attacker control.
  • OpenAI — Predicting model behavior before release by simulating deployment — replays ~1.3M de-identified real conversations through a candidate model; ~1.5x median multiplicative error; classic benchmarks flagged as tests almost 100% of the time vs ~5% for replayed real traffic.
  • Kairos: A Native World Model Stack for Physical AI — native pretraining stack blending open-world video, human behavioral data, and robot interaction across embodiments, aimed at persistent state and real-time execution.

Releases

  • Cursor Origin — Git-compatible, MCP-extensible code-hosting platform built for AI agents; demoed 22.6 commits/sec into a single repo; GA targeted for fall 2026. (Announced June 16 at Cursor's Compile conference.)
  • Anthropic opens Seoul office + Korean partnerships — third Asia-Pacific office (after Tokyo, Bengaluru); NAVER (Claude Code across engineering), Samsung SDS, LG CNS deployments; MOU with Korea's Ministry of Science and ICT.
  • NVIDIA Blackwell tops MLPerf Training 6.0 — fastest time to train on all seven benchmarks; GB300 NVL72 up to 1.6x over GB200; new DeepSeek-V3 671B MoE pretraining benchmark.
  • Android 17 with AppFunctions / on-device MCP — apps expose actions as tools for on-device agents like Gemini.
  • Qualcomm announces Snapdragon Reality Elite (MR glasses) and an AI-wearable turnkey kit.
  • llama.cpp adds model management (download/swap) via its API.
