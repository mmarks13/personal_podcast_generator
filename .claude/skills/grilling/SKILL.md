---
name: grilling
description: Interview the user relentlessly about a plan or design. Use when the user wants to stress-test a plan before building, or uses any 'grill' trigger phrases.
---

Interview me relentlessly about every aspect of this plan until we reach a shared understanding. Walk down each branch of the design tree, resolving dependencies between decisions one-by-one. You must ask every question through the AskUserQuestion tool — never in free-form prose — and for each question provide your recommended answer.

If a question can be answered by exploring the codebase, explore the codebase instead.
